### 🌐 [GMRoute](https://github.com/GrishaMelixov/GMRoute)
Интеллектуальный SOCKS5-прокси с маршрутизацией трафика по доменным правилам

- Trie-маршрутизация — 48 нс/op на 500 правилах

- 3D-дашборд с глобусом (WebGL)

### ☁️ [CloudMesh](https://github.com/GrishaMelixov/CloudMesh)
Распределённая платформа сбора и аналитики логов — 17+ млн событий в сутки

**Ключевые решения:**
- 4 Go-микросервиса: producer → collector → aggregator → api
- gRPC + Protobuf батчинг 500 событий/RPC (~5x vs REST)
- Kafka 12 партиций, ClickHouse OLAP + матвью (~10x быстрее)
- RabbitMQ topic exchange для алертинга по error rate
- Sliding window error monitor + дебаунс алертов
- K8s HPA autoscaling + GitHub Actions CI/CD

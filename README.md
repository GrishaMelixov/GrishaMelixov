### 🌐 [GMRoute](https://github.com/GrishaMelixov/GMRoute)
Интеллектуальный SOCKS5-прокси с маршрутизацией трафика по доменным правилам

**Ключевые решения:**
- Trie-маршрутизация — 48 нс/op на 500 правилах
- SNI-сниффинг из TLS ClientHello
- Zero-copy туннель через `sync.Pool`
- 3D-дашборд с глобусом (WebGL)
- 10 000+ concurrent TCP connections
